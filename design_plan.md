# Suryaroshni Website Redesign Plan
Based on Wipro Lighting Analysis

## Key Design Elements from Wipro:
1. **Primary Color**: Deep Purple (#351A55) - professional and modern
2. **Brand Theme**: Consistent purple across all elements
3. **Typography**: Clean, professional fonts (DM Sans)
4. **Layout**: Clean, corporate structure
5. **Hero Sections**: Bold headlines with professional imagery

## Planned Changes:

### 1. Color Scheme Update
- Primary: Deep Purple (#351A55) → Professional Purple
- Secondary: Electric Blue → Cool Blue accents
- Accent: Warm Amber → Gold/Orange highlights
- Background: Clean whites with subtle gradients

### 2. Typography Enhancement
- Update to professional font stack
- Improve hierarchy and spacing
- Better readability

### 3. Visual Improvements
- Enhanced gradients and lighting effects
- More sophisticated animations
- Better card designs
- Professional imagery integration

### 4. Layout Refinements
- Better spacing and padding
- Improved mobile responsiveness
- Professional contact sections
- Enhanced call-to-action buttons

## Implementation Steps:
1. Update global CSS with new color scheme
2. Enhance the main page layout and styling
3. Improve typography and spacing
4. Add professional animations
5. Test and refine
